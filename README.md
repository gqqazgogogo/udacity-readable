# Readable简介

服务端使用udacity提供的起始程序,前端通过create-react-app脚手架构建

## 启动项目

开始开发需要安装API server和frontend的依赖

* 安装启动服务端,需在终端下执行如下命令:
    - 进入服务端目录:`cd api-server`
    - 安装包依赖:`npm install`
    - 启动服务:`node server`
* 启动前端项目，需打开新的终端执行如下命令:
    - 进入前端工程目录:`cd frontend`
    - 安装包依赖:`npm install`
    - 启动react`npm start`
* 项目使用scss，如果修改样式，需打开新的终端执行如下命令:
    - 进入前端工程目录:`cd frontend`
    - 启动scss文件监听服务:`npm run watch-css`
